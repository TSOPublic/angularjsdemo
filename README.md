# Angular JS Demo
Angular JS Demo Application

## Environment Setup
### Dependencies
* install [git](https://git-scm.com/)
* install [node](https://nodejs.org/)
* install [gulp](http://gulpjs.com/) with `npm install gulp -g` (global install)

## Run
* run `npm install`
* run `node server.js`
* navigate to `localhost:2222`
